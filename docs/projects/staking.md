# Kokoro Staking

Repository: [https://github.com/happykokoro/kokoro-staking](https://github.com/happykokoro/kokoro-staking)

Live: [https://staking.happykokoro.com](https://staking.happykokoro.com)

Kokoro Staking is a multi-chain staking aggregator that unifies Proof-of-Stake staking across 17 blockchain networks into a single interface. Instead of using 17 separate chain-specific tools, users can stake tokens, explore validators, track rewards, and manage their staking positions from one dashboard.

## What it does

- **Staking** across 17 PoS chains through a unified interface
- **Validator explorer** with commission rates, uptime, delegation totals, and performance history
- **Reward tracking** across all chains in one view, with historical accrual and pending rewards
- **Staking calculator** using current network parameters (base APR, validator commission, compounding frequency) to project returns
- **Alerts** for staking events: reward payouts, validator status changes, slashing events, unbonding completions

## Supported chains

Ethereum, Solana, Cosmos, Polkadot, Cardano, Avalanche, Tezos, Near, Aptos, Sui, Celestia, Osmosis, Injective, Sei, Algorand, Mina, Polygon.

## Status

Deployed and running at [staking.happykokoro.com](https://staking.happykokoro.com). Under active development -- features, supported chains, and the interface are subject to change.
