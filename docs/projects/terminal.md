# Kokoro Terminal

Repository: [https://gitlab.com/kokoro-tech/products/kokoro-terminal](https://gitlab.com/kokoro-tech/products/kokoro-terminal)

Kokoro Terminal is a WebGPU-based terminal for displaying and interacting with crypto market data. It is the primary active project at Kokoro Tech and is currently under development.

## What it aims to do

- Real-time rendering of crypto market data using WebGPU
- Interactive charting and data visualization
- Integration with upstream market data sources

## Status

Under active development. This is the primary focus of current engineering work at Kokoro Tech. The codebase is changing frequently and is not ready for use by anyone outside the team.
