# Kokoro Tech

All Kokoro Tech projects are currently under development and have not been individually validated.

The previous versions of this site described a portfolio of products, architecture metrics, deployment status, and commercial services. That content has been removed because those claims were not individually verified. We will re-publish descriptions of specific projects only after each one has been validated on its own merits.

For now, the only actively developed project is [kokoro-terminal](https://github.com/happykokoro/kokoro-terminal), and even that is still in development.
