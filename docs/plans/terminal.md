## 1. Executive Summary

Kokoro Terminal is a professional-grade crypto market data terminal competing directly with established products like MMT (mmt.gg). It combines real-time multi-exchange data visualization, quantitative intelligence from the Kokoro Alpha Lab signal pipeline, trade execution, and an AI copilot — all in a single browser-based interface.

The terminal is the parent product of the Kokoro ecosystem. While Alpha Lab, wallet monitor, pricing service, and liquidation bot are specialized backend services, the terminal is what end users interact with and pay for. It surfaces the intelligence generated by the entire Kokoro infrastructure stack through a professional trading interface.

The product is built as a TypeScript + Bun monorepo with Rust-powered NAPI compute modules for performance-critical operations (footprint charts, heatmaps, TPO profiles, volume delta, VPIN computation, order book aggregation, and technical indicators). This hybrid architecture delivers institutional-grade computation speed in a web-native deployment model.

## 2. Product Overview

### What It Does

Kokoro Terminal provides professional traders with real-time market data from multiple cryptocurrency exchanges, advanced charting and visualization tools, quantitative signal intelligence, and direct trade execution — all in one integrated interface.

### Architecture

The terminal is structured as a monorepo with clear separation between frontend, API gateway, data ingestion, and compute modules:

- **Frontend** — Next.js with WebGL/Canvas for high-performance charting and 72+ React components covering every aspect of professional trading workflows
- **API Gateway** — Rust/Axum public REST + WebSocket server handling client connections and data distribution
- **Data Ingestion** — Rust service with exchange adapters for real-time market data from multiple sources
- **NAPI Compute Modules** — 10 Rust packages compiled to native Node.js addons via NAPI for performance-critical calculations:
  - `footprint-compute` — Footprint chart aggregation
  - `heatmap-compute` — Order book heatmap rendering
  - `orderbook-engine` — Real-time order book aggregation and management
  - `ta-indicators` — Technical analysis indicator library
  - `tpo-engine` — Time Price Opportunity (Market Profile) computation
  - `volume-delta` — Volume delta and cumulative delta analysis
  - `volume-profile` — Volume profile (POC, VAH, VAL) computation
  - `vpin-compute` — Volume-Synchronized Probability of Informed Trading
  - `api-client` — Typed API client shared across services
  - `shared-types` — Cross-package type definitions

### Intelligence Integration

The terminal integrates with the Kokoro Alpha Lab backend (64 Rust crates, 11 DSP filters, 24 alpha factors) for quantitative signal intelligence. Lab MCP provides 98 AI tool interfaces for copilot-driven analysis. The liquidation bot, wallet monitor, and pricing service all feed data into the terminal's intelligence layer.

### Frontend Components (72+)

The component library covers professional trading workflows: candlestick charts, footprint charts, heatmaps, order book visualization, DOM widgets, volume profiles, TPO/market profiles, liquidation maps, CVD panels, options Greeks, factor scores, backtest panels, alert configuration, copilot interface, billing, and portfolio management.

## 3. Competitive Positioning

### vs. MMT (mmt.gg)

MMT is the primary competitive benchmark. Kokoro Terminal matches MMT's core functionality (multi-exchange data, professional charting, execution) while adding capabilities MMT does not offer:

- **Quantitative signal intelligence** — 11 DSP filters and 24 alpha factors from Alpha Lab, surfaced directly in the terminal
- **AI copilot** — Natural language interaction with market data and trading tools via 98 MCP tool interfaces
- **On-chain data** — Wallet monitoring, smart money tracking, and DEX price aggregation from dedicated microservices
- **Multi-asset coverage** — Crypto, equities (Alpaca), forex, and prediction markets (Polymarket) through a single interface
- **Self-hosted option** — Docker Compose deployment for privacy-conscious professional traders

### vs. TradingView

TradingView targets a broader audience with social features and community scripts. Kokoro Terminal targets professional crypto traders who need deeper market microstructure data (footprint charts, VPIN, volume delta) and quantitative intelligence that TradingView does not provide.
