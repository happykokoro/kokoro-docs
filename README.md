- **Resume Profile** — Professional summary, timeline, achievements

## Tech Stack

Built with [MkDocs Material](https://squidfunk.github.io/mkdocs-material/) — dark theme, full-text search, tabbed navigation.

## Development

```bash
# Install
pip3 install mkdocs-material

# Local preview
mkdocs serve

# Deploy to GitHub Pages
mkdocs gh-deploy
```
